# Security Policy

## Supported Versions

Security fixes are provided on a best-effort basis for the latest published
release on [crates.io](https://crates.io/crates/fp_rust).

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| older   | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities privately rather than opening a public
GitHub issue.

**Preferred contact:** [JunYi JohnTeee Lee](mailto:johnteee@gmail.com)

You may also use [GitHub Security Advisories](https://github.com/TeaEntityLab/fpRust/security/advisories/new)
for the [TeaEntityLab/fpRust](https://github.com/TeaEntityLab/fpRust) repository.

Include as much detail as possible: affected version, reproduction steps, impact,
and any suggested fix. You should receive an acknowledgement within a reasonable
time; coordinated disclosure timelines will be agreed on a case-by-case basis.
